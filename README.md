## Кейс-задача № 4

**Условие:** Дано N голов драконов. У каждого дракона от 1 до 7 голов. Нужно найти максимальную силу драконьей стаи (произведение количества голов всех драконов).

**Решение:** `dragon_power.py` и `test_dragon_power.py`

### Алгоритм решения задачи № 4

Оптимальная стратегия:
1. Использовать максимальное количество драконов с 3 головами
2. Если остаток от деления N на 3 равен 1, заменяем 3+1 на 2+2
3. Если остаток равен 2, просто добавляем дракона с 2 головами
