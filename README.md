# Сумма отрицательных элементов между максимумом и минимумом

## Описание задачи
Дан одномерный массив A размерности N. 
Найти сумму отрицательных элементов, расположенных между максимальным и минимальным.

## Алгоритм
1. Найти индексы максимального и минимального элементов
2. Определить левую и правую границы интервала
3. Просуммировать отрицательные элементы на этом интервале
